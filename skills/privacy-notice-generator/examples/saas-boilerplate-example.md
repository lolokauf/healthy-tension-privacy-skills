# Privacy Notice Generator — SaaS Boilerplate Example

A worked example demonstrating the privacy-notice-generator skill applied to a SaaS boilerplate/starter kit with authentication, Stripe billing, email marketing, and AI features. Demonstrates standalone mode (no data inventory available).

---

## Input Description

A full-stack SaaS boilerplate (React + Node.js/Express + PostgreSQL) with:

- **User accounts:** Email/password + Google/GitHub OAuth, profile with name, email, avatar URL
- **Subscription billing:** Stripe integration (checkout, customer portal, webhooks for subscription events)
- **Email marketing:** Mailchimp/SendGrid integration for newsletter signups and drip campaigns
- **AI features:** OpenAI API integration for text generation, user prompts stored in database
- **Admin dashboard:** Admin users can view user list, subscription status, and usage statistics
- **File uploads:** User avatars and project files stored on AWS S3
- **Analytics:** Plausible Analytics (self-hosted, cookieless) + Sentry for error tracking
- **Background jobs:** Redis-based queue (BullMQ) for email sending, AI processing, webhook handling
- **API:** REST API with JWT authentication, rate limiting
- **Landing page:** Marketing site with newsletter signup form, testimonials
- **No data inventory file** — standalone mode (code scan only)
- **No data export endpoint** — users can view profile but cannot download all data
- **No account deletion** — soft delete with `deletedAt` column, no hard delete cron
